有趣的Telegram机器人们

2018/1/19 posted in  分享


Telegram是国外常用的聊天功能，和国内我们常用的QQ、微信功能相似。当然Telegram不能抢红包 在大陆访问和使用Telegram是一门技术活
Telegram比QQ和微信好玩一点是Telegram里有聊天机器人可以通过官方提供的api发送消息，视频，声音，文件等功能。这里给大家整理里一些Telegram中好评度较高的机器人。

https://t.me/BotFather

所有机器人的god 了，Telegram 官方管理；可以用它创建机器人和管理机器人，你可以利用机器人 API，帮你自己很多事情

https://t.me/IFTTT

ifttt官方机器人，可以很方便的和ifttt 里的其他服务连接，比如订阅某个网站的RSS，如果有更新就会自动把RSS内容推送到这个机器人，减少了频繁刷新网站来获取新的内容

https://t.me/scihubot

在聊天框中把你需要下载的文献名或者链接发给机器人，然后你就会收到回复

https://t.me/pronunciationbot

可以将84种语言文字转换成对应的语音 学习外语发音的利器

https://t.me/vote

投票机器人，新建一个投票发布在群里，每一次投票都是实时显示结果的，结果会以十分直观的方式展示

https://t.me/PTRankBot

可以帮助你搜索和分享iOS/macOS 应用，只需要在聊天框输入 @PTRankBot 和关键字，等待几秒就会出现搜索结果。支持命令查看 Store 各种排名，如 /top_free 查看每周免费排行榜

https://t.me/EFBSupport

ehForwarderBot能够将微信和Telegram 连接在一起，你可以在 Telegram 上收发微信消息，详情查看：http://www.appinn.com/eh-forwarder-bot
GitHub 地址：https://github.com/blueset/ehForwarderBot

https://t.me/AirPollution_Bot

查询空气质量(AQI)，数据来源为aqicn.org，比如发送“/aqi 成都”就会立即返回你成都的空气质量

https://t.me/temp_mail_bot

有效期10分钟临时邮箱服务，当你想要临时注册一个网站的账号时就可以用这种临时邮箱服务，避免自己的邮箱泄露也免于各种垃圾广告骚扰

https://t.me/GmailBot

可以在这里就方便的收发Gmail，Telegram 官方认证

https://t.me/utubebot

YouTube 视频下载或者音频转换服务，把YouTube的链接发过去它就会帮你转换格式并为你提供文件，便于你下载此视频

https://t.me/apkdl_bot

Android APK 搜索和下载，输入App 名即可提供多个来自国外网站APK下载链接，点Preview 可以查看图标

https://t.me/Instasave_bot

下载Instagram 和YouTube 的图片和视频

https://t.me/RustRssBot
https://t.me/superfeedr_bot
https://t.me/TheFeedReaderBot

这三个都是RSS 机器人

https://t.me/socks5_bot

免费获得Socks5 代理

https://t.me/mdrobot

Markdown 机器人

https://t.me/referbot

提供短网址 ShortenURL 服务，使用的网址为ume.la

https://t.me/thesafebot https://t.me/MyTeleCloudBot

这两个机器人都可以提供云存储服务
第一个利用 telegram 存储功能实现额外加密的云存储服务
第二个云存储服务，没有额外加密，不过功能还行

https://t.me/TransferRobot

上传文件后提供下载链接，适合做图床和文件分享
