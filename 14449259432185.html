双十一优惠券自动领取

2015/10/16 posted in  分享

双十一就要到了 淘宝也开始了各种预热活动，天猫上也开始了提前抢购／券的活动，参与商家不少，优惠券有划算的，但是估计数量不多。因为这点，就会出现一个问题：

手动抢太浪费时间了

应该有办法帮助我们自动抢的

如果你的电脑上有Chrome浏览器/Chrome内核的浏览器：比如QQ浏览器／360极速版浏览器


更多关于浏览器的知识

你需要：
1. 登录天猫双十一抢券页面：http://www.tmall.com/wow/act/14931/1111
2. 打开你的浏览器的Console模式

Window平台相关浏览器：

control+shift+J

Mac上Chrome谷歌浏览器

command+option+J

截图是Mac上的Chrome浏览器开启Console模式之后的界面：

直接复制下面这些代码：

(function(window, document, undefined) {
    var interval = 800;
    var closeDelay = 200;
    var index = 0;
    var couponLinks;
    var getCoupon = function() {
        if (index >= couponLinks.length) {
            console.log("领取完毕");
            return;
        }
        var coponLink = couponLinks[index];
        coponLink.click(); index++;
        console.log("领取 第" + index + " 张");
        setTimeout(getCoupon, interval);
        setTimeout(function() {
            var close = document.querySelector('.mui-dialog-close');
            if (close != null) close.click();       
        }, closeDelay);
    }
    var _scrollTop = 0;
    var _scrollStep = document.documentElement.clientHeight;
    var _maxScrollTop = document.body.clientHeight - document.documentElement.clientHeight;
    var autoScrollDown = setInterval(function() {
        _scrollTop += _scrollStep;
        if (_scrollTop > _maxScrollTop) {
            clearInterval(autoScrollDown);
            couponLinks = document.querySelectorAll('.mui-act-item-yhqbtn');
            console.log("总共：" + couponLinks.length + "条张优惠券待领取...");
            getCoupon();
        } else {
            document.body.scrollTop = _scrollTop;
        }
    }, 500);
}) (window, document);

黏贴完代码之后回车即可，接下来抢券的任务就可以放心交给代码来完成了。

视频操作：

如果你想多抢一些券来备战双十一的话，可以刷新抢券页面继续使用代码来抢券。
